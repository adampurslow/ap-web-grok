# Adam Purslow — Personal Site

This is the new personal website for Adam Purslow, replacing the previous WordPress site at [adampurslow.com](https://adampurslow.com).

## Current Status

**Stack:** Static HTML + Tailwind CSS (via CDN)  
**Design:** Clean, minimal, professional with a solid black navigation bar  
**Goal:** Personal brand + lead generation

This is the initial version built while testing the Grok + Vercel workflow.

## Local Development

```bash
cd ~/Desktop/Grok/projects/adampurslow.com

# Quick preview
npx serve .

# Or using Python
python3 -m http.server 8000
```
